# Modern Login Page UI

## Overview

A modern and responsive Login Page UI developed as part of the Synent Technologies Web Development Internship.

## Features

* Email Input Field
* Password Input Field
* Show/Hide Password Functionality
* Remember Me Checkbox
* Forgot Password Link
* Create Account Link
* Responsive Design
* Modern Dark Theme UI

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

```text
Task3_Login_UI/
│
├── index.html
├── style.css
└── README.md
```

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your web browser.
